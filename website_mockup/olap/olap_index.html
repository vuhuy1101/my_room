Online Analytical Processing

(OLAP)

OLAP is a computer processing that enables a user to easily extract and view data from different points of view. OLAP data is stored in a multidimensional database, in this case the three dimensions are Product, Time, and Store. OLAP is really useful for analyzing data, figuring out the relationship between data items, and can be used for data mining as well. For example, a store owner can use OLAP to display his store revenues over each month, how many products were sold over the summer, or simple compare the revenues of different stores over the same period.

I worked with a group of 3 to build a simple web application showing how OLAP operations work (including slice, dice, roll-up, and drill-down). I was a team lead who designed the front-end, part of the back-end, and made sure that the team was on the same page by keeping up the team communication. For the project, we decided to use Python microframework Flask because it is simple and easy to setup. For the front-end, we used Javascript and jQuery together with Bootstrap to update the data table after each operation without refreshing the page.
